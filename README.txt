Clone do YouTube

Esse projeto pretende fazer uma cópia do site YouTube, tentando recriar apenas
os aspectos de layout. As ferramentas usadas foram apenas HTML e CSS.

O site está hospedado na plataforma Vercel e 
pode ser acessado através do link: https://youtube-clone-navy-mu.vercel.app/

Todo o projeto foi inspirado e baseado no criador de conteúdo SuperSimpleDev.

Link para do canal SuperSimpleDev: https://www.youtube.com/@SuperSimpleDev
Link para o vídeo que ensina a desenvolver o projeto: https://www.youtube.com/watch?v=G3e-cpL7ofc
Todo o conteúdo original pertence ao criador SuperSimpleDev. 
Esse projeto foi desenvolvido apenas para fins educacionais.
